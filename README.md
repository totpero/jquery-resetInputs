jquery-resetInputs
==================

JQuery plugin for reset inputs in selector

## Demo
http://www.totpe.ro/

##How to use:
Call the plugin on the element/elements/form that you want to clear/reset.

##Options:
```javascript
	$(selector).resetInputs({
		'ignor': '#div'
	});
	$('elem').resetInputs();
```
